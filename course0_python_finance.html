Section 1: Introduction to Programming

1.1 What is Programming?

Imagine you're a trader in the 1980s. To calculate a 20-day moving average, you'd grab a calculator, punch in 20 closing prices, add them up, divide by 20, write down the result, then repeat this process hundreds of times for each stock you're tracking.

Programming is simply writing instructions for a computer to follow.

Think of it like a recipe: - A recipe tells a cook exactly what steps to follow - A program tells a computer exactly what operations to perform

The difference? A computer can follow your instructions millions of times per second, never gets tired, and never makes arithmetic mistakes.

ğŸ“ˆ Finance Note: Today, with just a few lines of Python, you can analyze decades of price data, screen thousands of stocks, and execute trades - all in the time it takes to blink.

1.2 Why Python for Finance?

Python has become the dominant language in finance for several reasons:

1. Readable Syntax Python reads almost like English, making it easier to learn and maintain.

2. Powerful Libraries - pandas - Data manipulation (used by every quant) - numpy - Fast numerical computing - matplotlib / plotly - Visualization - yfinance - Free market data

3. Industry Adoption - Goldman Sachs, JPMorgan, and most hedge funds use Python - Bloomberg terminals support Python - Most trading APIs have Python libraries

Python reads and executes code line by line, from top to bottom. This is important to understand because the order of your instructions matters.

The output appears in exactly the order we wrote the instructions. This sequential execution is fundamental to how programs work.

Section 2: Variables & Assignment

2.1 What are Variables?

A variable is like a labeled container that holds a value. Instead of typing 178.50 every time you need Apple's stock price, you can store it in a variable called aapl_price and use that name instead.

Think of variables like labeled boxes in a warehouse: - The label (variable name) tells you what's inside - The contents (value) can change over time

ğŸ“ˆ Finance Note: Variables are essential in trading. You might store entry_price, stop_loss, position_size, and target_price as variables, then use them in your trading calculations.

2.2 Creating Variables

In Python, we create variables using the assignment operator =:

        logger.info(f'Alert added: {ticker} {condition} ${target}')

    def check_alerts(self):
        for alert in self.alerts:
            if not alert['active']:
                continue
            price = self.prices.get(alert['ticker'])
            triggered = False
            if alert['condition'] == 'above' and price > alert['target']:
                triggered = True
            elif alert['condition'] == 'below' and price < alert['target']:
                triggered = True
            if triggered:
                alert['active'] = False
                logger.warning(f\"ğŸš¨ {alert['ticker']} {alert['condition']} ${alert['target']:.2f}!\")

    def run(self, iterations=5):
        logger.info('Starting alert system')
        for i in range(iterations):
            self.update_prices()
            self.check_alerts()

class DataManager:
    def __init__(self):
        self.data = {}

    def generate_data(self, ticker, days=252, start_price=100):
        dates = pd.date_range(end=pd.Timestamp.today(), periods=days, freq='B')
        returns = np.random.normal(0.0005, 0.02, days)
        close = start_price * np.cumprod(1 + returns)

        df = pd.DataFrame({
            'Open': close * (1 + np.random.uniform(-0.01, 0.01, days)),
            'High': close * (1 + np.abs(np.random.normal(0, 0.015, days))),
            'Low': close * (1 - np.abs(np.random.normal(0, 0.015, days))),
            'Close': close,
            'Volume': np.random.randint(1_000_000, 10_000_000, days)
        }, index=dates)

        self.data[ticker] = df
        return df

    def add_indicators(self, ticker):
        df = self.data[ticker].copy()
        df['SMA_20'] = df['Close'].rolling(20).mean()
        df['SMA_50'] = df['Close'].rolling(50).mean()
        df['RSI'] = self._calculate_rsi(df['Close'])
        df['Returns'] = df['Close'].pct_change()
        self.data[ticker] = df
        return df

    def _calculate_rsi(self, prices, period=14):
        delta = prices.diff()
        gain = (delta.where(delta > 0, 0)).rolling(period).mean()
        loss = (-delta.where(delta < 0, 0)).rolling(period).mean()
        rs = gain / loss
        return 100 - (100 / (1 + rs))

class Strategy(ABC):
    @abstractmethod
    def generate_signals(self, data):
        pass

class MACrossStrategy(Strategy):
    def __init__(self, fast=20, slow=50):
        self.fast = fast
        self.slow = slow
        self.name = f'MA_Cross_{fast}_{slow}'

    def generate_signals(self, data):
        df = data.copy()
        df['Signal'] = 0
        df.loc[df[f'SMA_{self.fast}'] > df[f'SMA_{self.slow}'], 'Signal'] = 1
        df.loc[df[f'SMA_{self.fast}'] < df[f'SMA_{self.slow}'], 'Signal'] = -1
        df['Position'] = df['Signal'].diff().fillna(0)
        return df

class RSIStrategy(Strategy):

class PortfolioManager:
    def __init__(self, initial_capital=100000, max_position=0.2):
        self.initial_capital = initial_capital
        self.cash = initial_capital
        self.positions = {}
        self.max_position = max_position
        self.trades = []

    def get_position_size(self, price):
        max_value = self.get_value() * self.max_position
        return int(max_value / price)

    def execute_trade(self, ticker, shares, price, date):
        if shares == 0:
            return None

        action = 'BUY' if shares > 0 else 'SELL'
        value = abs(shares) * price

        if action == 'BUY':
            if value > self.cash:
                shares = int(self.cash / price)
                value = shares * price
            if shares == 0:
                return None
            self.cash -= value
            self.positions[ticker] = self.positions.get(ticker, 0) + shares
        else:
            shares = min(abs(shares), self.positions.get(ticker, 0))
            value = shares * price
            self.cash += value
            self.positions[ticker] -= shares
            if self.positions[ticker] == 0:
                del self.positions[ticker]

        self.trades.append({'date': date, 'ticker': ticker, 'action': action, 'shares': shares, 'price': price})
        return {'action': action, 'shares': shares}

    def get_value(self, prices=None):
        pos_value = 0
        if prices:
            for ticker, shares in self.positions.items():
                pos_value += shares * prices.get(ticker, 0)
        return self.cash + pos_value

class Backtester:
    def __init__(self, data_manager, initial_capital=100000):
        self.dm = data_manager
        self.initial_capital = initial_capital
        self.results = None

    def run(self, ticker, strategy):
        data = self.dm.data[ticker]
        signals = strategy.generate_signals(data)

        portfolio = PortfolioManager(self.initial_capital)
        values = []

        for date, row in signals.iterrows():
            price = row['Close']
            pos_change = row['Position']

            if pos_change > 0:
                shares = portfolio.get_position_size(price)
                portfolio.execute_trade(ticker, shares, price, date)
            elif pos_change < 0 and ticker in portfolio.positions:
                shares = portfolio.positions[ticker]
                portfolio.execute_trade(ticker, -shares, price, date)

            values.append({'date': date, 'value': portfolio.get_value({ticker: price})})

        results = pd.DataFrame(values).set_index('date')
        results['returns'] = results['value'].pct_change()

        total_return = (results['value'].iloc[-1] / self.initial_capital - 1) * 100
        sharpe = results['returns'].mean() / results['returns'].std() * np.sqrt(252) if results['returns'].std() > 0 else 0
        cummax = results['value'].cummax()
        drawdown = (results['value'] - cummax) / cummax
        max_dd = drawdown.min() * 100

        self.results = {
            'data': results,
            'metrics': {'total_return': total_return, 'sharpe': sharpe, 'max_drawdown': max_dd, 'trades': len(portfolio.trades)},
            'strategy': strategy.name
        }
        return self.results

    def plot(self):
        if not self.results:
            return
        fig, axes = plt.subplots(2, 1, figsize=(12, 8))
        axes[0].plot(self.results['data']['value'])
        axes[0].axhline(self.initial_capital, linestyle='--', color='gray')
        axes[0].set_title(f\"{self.results['strategy']} - Portfolio Value\")
        cummax = self.results['data']['value'].cummax()
        dd = (self.results['data']['value'] - cummax) / cummax * 100
        axes[1].fill_between(dd.index, dd, color='red', alpha=0.3)
        axes[1].set_title('Drawdown (%)')
        plt.tight_layout()
        plt.show()
